Detailed Description

The APM2000_Inventory class Automatic Pencil Machine(APM)
Suffix "2000" indicates 2000-series. Every 2000-series produces 2 pencils per second, which are automatically added into the inventory. APM stops automatically when there are insufficient materials and resumes when materials are available.
