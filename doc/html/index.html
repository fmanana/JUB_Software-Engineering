Pencil Producer

Introduction

This project implements a pencil producer game where the player can produce and sell pencils to the public. In order to produce pencils the player must buy materials (wood, graphite). The player can increase and decrease the price of pencils at any point, which decreases and increases the demand for pencils respectively.

Compilation

$ cd se-02-team/Pencil-Producer/
$ mkdir build
$ cd build/
$ cmake .. & make
$ ./pencilproducer
